# Employee Attrition Analysis

## Objective
Analyze employee attrition patterns and identify key factors contributing to employee turnover using exploratory data analysis (EDA).

## Dataset
IBM HR Employee Attrition dataset containing employee demographics, job details, and attrition status.

## Tools Used
- Python
- pandas
- matplotlib
- Jupyter Notebook

## Key Insights
- Attrition is highest among employees in early tenure (0â€“2 years).
- Attrition decreases steadily as tenure increases.
- Long-tenured employees show strong retention and stability.

## Conclusion
Early-stage employee retention appears to be a critical area for reducing overall attrition. Targeted onboarding and engagement strategies may help mitigate early churn.
